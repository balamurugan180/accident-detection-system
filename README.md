
## Dataset Information
The dataset used for this project is sourced from Kaggle:  
[Accident Detection from CCTV Footage](https://www.kaggle.com/datasets/ckay16/accident-detection-from-cctv-footage)


Step 1: Clone the Project
bashgit clone https://github.com/YOUR_USERNAME/accident-detection-system.git
cd accident-detection-system

Step 2: Create Virtual Environment
bash# Windows
python -m venv venv

# Activate it
venv\Scripts\activate

Step 3: Install Requirements
bashpip install -r requirements.txt

Step 4: Create Missing Folders
bash# Windows
mkdir static\uploads
mkdir static\results

Step 5: Download Model Files

⚠️ These files are not in GitHub (too large)



Download model_weights.h5 → [Google Drive Link]
Place both files in the root folder


Step 6: Run the Project
bashpython app.py
Step 7: Open Browser
http://127.0.0.1:5000
